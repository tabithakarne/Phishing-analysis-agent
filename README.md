# Phishing-analysis-agent
AI powered phishing email analysis agent 
## Purpose
This project demonstrates how an AI-powered phishing analysis agent can be built
using structured prompts instead of traditional programming.

The agent analyzes email content to:
- Detect phishing indicators
- Assign a risk score
- Classify the email
- Explain the reasoning
- Recommend user actions

## Key Skills Demonstrated
- Cybersecurity threat analysis
- Phishing detection heuristics
- AI prompt engineering
- SOC-style documentation
- Risk-based decision making

## Target Users
- SOC Analysts
- Security Awareness Teams
- Cybersecurity Interns
